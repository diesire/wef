h2. Features
* Educational purposes. Because I`m learning JavaScript and Hello, world! is mainstream.
* Work in progress. You know, forever beta stage.
* Eating your own dog food. Wef is used in my awesome "CSS Template Layout Module":https://github.com/diesire/cssTemplateLayout

* Modular, extensible
